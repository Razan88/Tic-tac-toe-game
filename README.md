## TIC TAC TOE GAME 
It's a simple Tic-Tac-Toe game for two players. Each player takes turns placing their character's head onto a 3x3 grid; whoever gets a straight line first wins the game.

![screenshot ](icon_0_small.png)

### Built With:
* html
* CSS
* JavaScrpit
* JQuery

### Challenges:
